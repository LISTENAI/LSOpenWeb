
# 硬件参考设计

### 原理图/PCBA设计

原理图设计6个模拟驻极体MIC或模拟硅麦设计，设计采用3颗ES7202 ADC获取MIC模拟信号（开发者需根据产品定义修改原理图调整为2MIC设计），内部模数转换后通过PDM接口连接到CSK，采用1颗ES7202 ADC做喇叭硬回采，采用一颗DAC ES7148/ES7149，将从CSK发出的标准IIS信号转换为模拟信号给到喇叭播音。
